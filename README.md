
## Techs

Next.js
React.js
TypeScript

## Requeriments

Git installed
Node installed
Npm or Yarn installed

## Getting Started

First, do a git clone of:

https://github.com/MiDrAjX/NLW4.0-Moveit-next.git

install the dependencies:

```bash
npm install
# or
yarn install
```

so, run the development server:

```bash
npm run dev
# or
yarn dev
```
